# 💻 A3 - Sistema de Desenvolvimento de Software

## 📌 Sobre o Projeto
Projeto desenvolvido na disciplina de Sistemas de Desenvolvimento de Software, com o objetivo de aplicar na prática conceitos de desenvolvimento, modelagem e programação orientada a objetos.

O sistema simula um cenário real de aplicação, focando na organização de código, lógica de negócio e boas práticas de desenvolvimento.

## 🎯 Objetivo
Desenvolver um sistema funcional aplicando conceitos de engenharia de software, simulando um ambiente real de desenvolvimento.
